## 🌃 الفكرة

في قلب القاهرة الكبرى المزدحم، حيث تتشابك خطوط المترو الحمراء والزرقاء والخضراء مع القطار الكهربائي الخفيف (LRT) ومونوريل العاصمة الإدارية الجديدة — ولدت **"محطتي"**.

ليس مجرد "تطبيق مواصلات"...  
هو **عقل إلكتروني** يفكر بدلاً منك، يحسب بدقة علمية، ويقدم لك **أفضل مسار متكامل** عبر كل شبكات النقل في مصر، مع تسعيرة موحدة وتفاصيل زمنية دقيقة.

> *"من حلوان للعاصمة الإدارية، من جامعة القاهرة للجامعة البريطانية بالشروق — نعرف الطريق."* 🧭

---

## ✨ ما يميز "محطتي"

| الميزة | التفاصيل |
|--------|----------|
| 🧠 **خوارزمية ديجسترا محسّنة** | تعتمد على **عقوبة زمنية ذكية** للتبديل بين الخطوط (5 دقائق)، تضمن أقل زمن رحلة وليس أقل محطات فقط |
| 🚇 **شبكة متكاملة 2026** | 3 خطوط مترو + 18 محطة LRT + 35 محطة مونوريل (شرق/غرب النيل) — كلها في خريطة واحدة |
| 💳 **تسعيرة موحدة ذكية** | حساب تلقائي لسعر التذكرة بناءً على عدد المحطات في كل نظام نقل (مترو مدعوم / LRT / مونوريل تقديري) |
| 🏛️ **دعم المعالم والجامعات** | اكتب "متحف التحرير" أو "الجامعة البريطانية" — التطبيق يحلها تلقائياً لأقرب محطة |
| 🎨 **واجهة Cyberpunk Glassmorphism** | تصميم نيون مستقبلي بتأثيرات 3D على التذكرة، وخلفية شبكية متحركة |
| 📱 **PWA كامل** | يعمل Offline بفضل Service Worker مع استراتيجية Stale-While-Revalidate |
| 🔍 **بحث فازي بالعربية** | تطبيع تلقائي للألفات والتاء المربوطة والتشكيل — يفهمك حتى لو أخطأت في الكتابة |
| 🎫 **تذكرة ذكية ثلاثية الأبعاد** | بطاقة تفاعلية تتبع حركة الفأرة (Tilt Effect) مع shimmer متحرك |

## 🗺️ الخريطة المتكاملة (يونيو 2026)

```
🔴 الخط الأول    : حلوان ←→ المرج الجديدة  (35 محطة)
🔵 الخط الثاني   : شبرا الخيمة ←→ المنيب    (20 محطة)
🟢 الخط الثالث   : عدلي منصور ←→ روض الفرج / جامعة القاهرة (32 محطة)
⚡ LRT           : المستقبل ←→ العاشر من رمضان / مطار العاصمة (18 محطة)
🟦 مونوريل شرق   : الاستاد ←→ مدينة العدل   (22 محطة)
🟩 مونوريل غرب   : المنطقة الصناعية ←→ وادي النيل (13 محطة — قيد الإنشاء)
```

## 🏗️ البنية التقنية

```
📁 mahataty-2026/
├── 📄 index.html          ← هيكل التطبيق (PWA Shell)
├── 🎨 style.css           ← تصميم Cyberpunk + Glassmorphism + Animations
├── ⚙️ app.js              ← منطق الواجهة والتفاعل
├── 🧭 router.js           ← خوارزمية المسار الأمثل (Dijkstra)
├── 🗂️ data.js             ← بيانات الشبكة المتكاملة (Graph)
├── 🏛️ landmarks.js        ← المعالم والجامعات ومواقف السفر
├── ⚡ config.js            ← التسعير وإعدادات التحويل
├── 📋 manifest.json       ← إعدادات PWA
└── 🔧 sw.js               ← Service Worker (Cache First)
```

### 🧮 خوارزمية المسار

```javascript
// Priority Queue-based Dijkstra مع عقوبة التبديل
// كل عقدة: { station, dist, line, transfers }
// التبديل في محطة تحويل = +5 دقائق (انتقال رصيف + انتظار)
// التبديل غير القانوني = +999 دقيقة (منع المسارات المستحيلة)
```

## 🚀 التشغيل المحلي

```bash
# 1. استنساخ المستودع
git clone https://github.com/seif4d/mahataty-2026.git

# 2. الدخول للمجلد
cd mahataty-2026

# 3. تشغيل خادم محلي (أي خادم ستاتيكي)
# Python 3
python -m http.server 8000
# أو Node.js
npx serve .
# أو PHP
php -S localhost:8000

# 4. افتح المتصفح
open http://localhost:8000
```

> 💡 **للحصول على أفضل تجربة:** استخدم Chrome/Safari على الجوال واضغط "Add to Home Screen" لتثبيت التطبيق.

## 🛠️ التقنيات المستخدمة

- **Vanilla JavaScript (ES6 Modules)** — بدون إطارات عمل
- **CSS Custom Properties & Grid/Flexbox** — تصميم متجاوب بالكامل
- **Web App Manifest** — PWA قابل للتثبيت
- **Service Worker** — Cache First + Stale-While-Revalidate
- **CSS 3D Transforms** — تأثير Tilt على بطاقة التذكرة
- **CSS Animations** — Shimmer, Pulse, Float, Grid Pulse
- **Arabic Text Normalization** — دعم كامل للبحث العربي
