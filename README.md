## 🌃 What's Mahataty?

> *"In the heart of Greater Cairo's chaos, where red, blue, and green metro lines weave through the LRT and the New Capital's monorail — Mahataty was born."*

It's **not just a transit app**... it's an **electronic brain** that thinks for you, calculates with scientific precision, and delivers the **optimal integrated route** across ALL of Egypt's transit networks — with a unified fare and accurate time estimates.

From **Helwan to the New Capital**, from **Cairo University to BUE in Shorouk** — we know the way. 🧭

## ✨ What Makes Mahataty Different?

| Feature | Details |
|---------|---------|
| 🧠 **Optimized Dijkstra** | Smart **5-minute transfer penalty** — guarantees the *fastest real-world route*, not just the shortest station count |
| 🚇 **Fully Integrated 2026 Network** | 3 Metro lines + 18 LRT stations + 35 Monorail stations — all in one unified graph |
| 💳 **Smart Unified Fare** | Auto-calculates fare based on station count per transit system (Metro subsidized / LRT / Monorail estimated) |
| 🏛️ **Landmark & University Support** | Type "Egyptian Museum" or "British University" — we resolve it to the nearest station automatically |
| 🎨 **Cyberpunk Glassmorphism UI** | Neon futuristic design with 3D ticket tilt effects, animated grid background |
| 📱 **Full PWA** | Works **completely offline** via Service Worker with Stale-While-Revalidate strategy |
| 🔍 **Fuzzy Arabic Search** | Auto-normalization of Alef variants, Ta Marbuta, and diacritics — understands you even with typos |
| 🎫 **3D Smart Ticket Card** | Interactive card that follows mouse movement (Tilt Effect) with animated shimmer |

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/seif4d/Mahatati.git

# 2. Enter directory
cd Mahatati

# 3. Start a local server (any static server works)
# Python 3
python -m http.server 8000
# Node.js
npx serve .
# PHP
php -S localhost:8000

# 4. Open in browser
open http://localhost:8000
```

> 💡 **Pro Tip:** Use Chrome/Safari on mobile and tap **"Add to Home Screen"** to install the PWA!

## 🏗️ Tech Stack

```
📁 Mahatati/
├── 📄 index.html          ← PWA Shell Structure
├── 🎨 style.css           ← Cyberpunk + Glassmorphism + GPU-Optimized Animations
├── ⚙️ js/app.js          ← UI Logic & PWA Install Handler
├── 🧭 js/router.js        ← Dijkstra Shortest Path Algorithm
├── 🗂️ js/data.js         ← Integrated Transit Network Graph
├── 🏛️ js/landmarks.js    ← Landmarks, Universities & Transit Hubs
├── ⚡ js/config.js        ← Fare Rules & Transfer Settings
├── 📋 manifest.json       ← PWA Manifest
└── 🔧 sw.js               ← Service Worker (Cache First)
```

### 🧮 The Algorithm

```javascript
// Priority Queue-based Dijkstra with Transfer Penalty
// Each node: { station, dist, line, transfers }
// Transfer at interchange station = +5 min (platform walk + wait)
// Illegal transfer (no shared platform) = +999 min (prevents impossible routes)
```

## 🌃 إيه فكرة "محطتي"؟

> *"في قلب زحمة القاهرة الكبرى، حيث تتشابك خطوط المترو الحمراء والزرقاء والخضراء مع الـ LRT ومونوريل العاصمة الإدارية الجديدة — ولدت "محطتي"."*

مش مجرد "تطبيق مواصلات"... ده **عقل إلكتروني** بيفكر بدالك، يحسب بدقة علمية، ويقدملك **أفضل مسار متكامل** عبر كل شبكات النقل في مصر، مع تسعيرة موحدة وتفاصيل زمنية دقيقة.

من **حلوان للعاصمة الإدارية**، من **جامعة القاهرة للجامعة البريطانية بالشروق** — نعرف الطريق. 🧭
